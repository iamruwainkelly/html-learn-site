Media Elements and Optimization

Images

Basic Image Element

<img src="path/to/image.jpg" alt="Description of image">

Image Attributes

  • src - Source URL of the image
  • alt - Alternative text description
  • width - Width in pixels
  • height - Height in pixels
  • loading - Loading behavior (eager/lazy)

Responsive Images

<picture>
  <source media="(min-width: 800px)" srcset="large.jpg">
  <source media="(min-width: 400px)" srcset="medium.jpg">
  <img src="small.jpg" alt="A responsive image">
</picture>

Audio Elements

Basic Audio Element

<audio src="music.mp3" controls></audio>

Audio with Multiple Sources

<audio controls>
  <source src="audio.ogg" type="audio/ogg">
  <source src="audio.mp3" type="audio/mpeg">
  Your browser does not support the audio element.
</audio>

Audio Attributes

  • controls - Show playback controls
  • autoplay - Start playing automatically
  • loop - Repeat when finished
  • muted - Start muted
  • preload - Loading behavior (auto/metadata/none)

Video Elements

Basic Video Element

<video src="video.mp4" controls width="640" height="360"></video>

Video with Multiple Sources

<video controls width="640" height="360">
  <source src="video.webm" type="video/webm">
  <source src="video.mp4" type="video/mp4">
  Your browser does not support the video element.
</video>

Video with Poster Image

<video 
  controls 
  width="640" 
  height="360"
  poster="thumbnail.jpg">
  <source src="video.mp4" type="video/mp4">
</video>

Media Optimization

Image Formats

  • JPEG/JPG - Best for photographs
  • PNG - Best for graphics with transparency
  • WebP - Modern format with better compression
  • SVG - Vector graphics that scale perfectly
  • AVIF - Next-generation image format

Best Practices

  • Choose appropriate image formats
  • Optimize file sizes
  • Use responsive images
  • Provide alt text for accessibility
  • Consider lazy loading for better performance

SVG Integration

Inline SVG

<svg width="100" height="100" viewBox="0 0 100 100">
  <circle cx="50" cy="50" r="40" stroke="black" fill="red"/>
</svg>

SVG as Image

<img src="icon.svg" alt="Icon description">

SVG as Background

<div style="background-image: url('pattern.svg')"></div>

Iframe Elements

Basic Iframe

<iframe 
  src="https://example.com" 
  width="600" 
  height="400" 
  title="Embedded content">
</iframe>

YouTube Video Embed

<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  title="YouTube video player"
  allow="accelerometer; autoplay; clipboard-write; 
         encrypted-media; gyroscope; picture-in-picture"
  allowfullscreen>
</iframe>
