Semantic HTML

Understanding Semantic HTML

Semantic HTML uses elements that carry meaning about the structure and content of a web page, making it more accessible and easier to understand for both humans and machines.

Document Structure Elements

  • <header>

    Contains introductory content, navigation, or heading elements

    <header>
  <h1>Site Title</h1>
  <nav>Site Navigation</nav>
</header>
  • <nav>

    Contains navigation links

    <nav>
  <ul>
    <li><a href="#">Home</a></li>
    <li><a href="#">About</a></li>
  </ul>
</nav>
  • <main>

    Contains the main content of the document

  • <article>

    Represents a self-contained composition

  • <section>

    Groups related content

  • <aside>

    Contains content tangentially related to the main content

  • <footer>

    Contains footer information for its nearest sectioning content

Text-Level Semantic Elements

  • <em>

    Emphasis: This text has emphasis

  • <strong>

    Strong importance: This text is strongly emphasized

  • <cite>

    Citation: The HTML Living Standard

  • <time>

    Time/Date: May 16, 2025

  • <abbr>

    Abbreviation: HTML

  • <address>

    Contact information for the nearest article or body element

Best Practices

  • Use headings (<h1> - <h6>) to create a logical document outline
  • Choose semantic elements over generic <div> elements when possible
  • Use <button> for clickable controls instead of styled <div> elements
  • Include proper ARIA attributes when semantic HTML isn't sufficient
  • Maintain a logical heading hierarchy (don't skip levels)

Example Page Structure

<body>
  <header>
    <h1>Website Title</h1>
    <nav>
      <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">About</a></li>
      </ul>
    </nav>
  </header>

  <main>
    <article>
      <h2>Article Title</h2>
      <p>Article content...</p>
      
      <section>
        <h3>Section Heading</h3>
        <p>Section content...</p>
      </section>
    </article>

    <aside>
      <h2>Related Content</h2>
      <ul>
        <li>Related link 1</li>
        <li>Related link 2</li>
      </ul>
    </aside>
  </main>

  <footer>
    <p>© 2025 Your Name</p>
  </footer>
</body>
