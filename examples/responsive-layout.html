Practice Exercise

Create a responsive product grid with the following requirements:

  • Grid should display 4 columns on desktop
  • 3 columns on tablet (min-width: 768px)
  • 2 columns on mobile (min-width: 480px)
  • 1 column on small mobile devices
  • Each product should have an image, title, price, and "Add to Cart" button
View Solution 
<style>
.product-grid {
  display: grid;
  gap: 1rem;
  grid-template-columns: repeat(4, 1fr);
}

.product-card {
  border: 1px solid #ddd;
  padding: 1rem;
  text-align: center;
}

.product-card img {
  width: 100%;
  height: auto;
}

@media (max-width: 1024px) {
  .product-grid {
    grid-template-columns: repeat(3, 1fr);
  }
}

@media (max-width: 768px) {
  .product-grid {
    grid-template-columns: repeat(2, 1fr);
  }
}

@media (max-width: 480px) {
  .product-grid {
    grid-template-columns: 1fr;
  }
}
</style>

<div class="product-grid">
  <div class="product-card">
    <img src="product1.jpg" alt="Product 1">
    <h3>Product Name</h3>
    <p class="price">$19.99</p>
    <button>Add to Cart</button>
  </div>
  <!-- More product cards... -->
</div>
