HTML Tables

Table Structure

Tables are used to display tabular data in rows and columns. A proper table structure includes:

  • <table> - Container for the table
  • <caption> - Table title/description
  • <thead> - Header section
  • <tbody> - Main content
  • <tfoot> - Footer section

Basic Table Elements

  • <tr> - Table row
  • <th> - Header cell
  • <td> - Data cell

Simple Table Example

<table>
  <tr>
    <th>Header 1</th>
    <th>Header 2</th>
  </tr>
  <tr>
    <td>Data 1</td>
    <td>Data 2</td>
  </tr>
</table>

Table Sections

<table>
  <caption>Monthly Sales Report</caption>
  
  <thead>
    <tr>
      <th>Month</th>
      <th>Sales</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>January</td>
      <td>$10,000</td>
    </tr>
    <tr>
      <td>February</td>
      <td>$12,000</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <th>Total</th>
      <td>$22,000</td>
    </tr>
  </tfoot>
</table>

Table Attributes

Cell Spanning

  • colspan - Span multiple columns
  • rowspan - Span multiple rows
<table>
  <tr>
    <th colspan="2">Name</th>
    <th>Age</th>
  </tr>
  <tr>
    <td>First</td>
    <td>Last</td>
    <td rowspan="2">25</td>
  </tr>
  <tr>
    <td colspan="2">Contact Info</td>
  </tr>
</table>

Accessibility Features

Scope Attribute

Use scope to associate header cells with their data cells:

<table>
  <tr>
    <th scope="col">Name</th>
    <th scope="col">Age</th>
  </tr>
  <tr>
    <th scope="row">John</th>
    <td>25</td>
  </tr>
</table>

Headers and ID Attributes

For complex tables, use headers and id to associate cells:

<table>
  <tr>
    <th id="name">Name</th>
    <th id="age">Age</th>
  </tr>
  <tr>
    <td headers="name">John</td>
    <td headers="age">25</td>
  </tr>
</table>

Complex Table Example

<table>
  <caption>Quarterly Sales Report</caption>
  
  <thead>
    <tr>
      <th scope="col">Region</th>
      <th scope="col">Q1</th>
      <th scope="col">Q2</th>
      <th scope="col">Q3</th>
      <th scope="col">Q4</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <th scope="row">North</th>
      <td>$10,000</td>
      <td>$12,000</td>
      <td>$15,000</td>
      <td>$18,000</td>
    </tr>
    <tr>
      <th scope="row">South</th>
      <td>$8,000</td>
      <td>$9,000</td>
      <td>$11,000</td>
      <td>$13,000</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <th scope="row">Total</th>
      <td>$18,000</td>
      <td>$21,000</td>
      <td>$26,000</td>
      <td>$31,000</td>
    </tr>
  </tfoot>
</table>
