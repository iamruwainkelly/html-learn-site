HTML Basics

Role of HTML and Basic Structure

HTML represents the content and structure of web pages. Elements are the building blocks for an HTML document, consisting of opening and closing tags:

<elementName>Content goes here</elementName>

Void Elements

Void elements cannot have content and only have a start tag. Examples include:

<img>
<meta>

Both of these syntaxes are acceptable:

<img>
<img/>

Common HTML Elements

Headings

HTML provides six levels of headings:

H1: Most important heading

H2: Second level heading

H3: Third level heading

H4: Fourth level heading

H5: Fifth level heading
H6: Least important heading

Text Elements

Common text elements include:

  • <p> - For paragraphs
  • <em> - For emphasis
  • <strong> - For strong importance

Lists

Unordered List:
  • First item
  • Second item
  • Third item
Ordered List:
  1. First step
  2. Second step
  3. Third step

Images and Media

Images are added using the img element:

<img src="image.jpg" alt="Description of image">

Figure with Caption

Links and Navigation

Links are created using the anchor (<a>) element:

<a href="https://example.com">Visit Example</a>

Link Target Types:

  • _self - Default, opens in same window
  • _blank - Opens in new window/tab
  • _parent - Opens in parent frame
  • _top - Opens in full window

Best Practices

  • Use semantic HTML elements
  • Include proper meta tags for SEO
  • Provide alt text for images
  • Use appropriate heading hierarchy
  • Ensure proper document structure
