HTML Best Practices

Document Structure

Always Include Essential Meta Tags

<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="utf-8">
    <meta name="viewport" content="width=device-width, initial-scale=1">
    <meta name="description" content="Page description">
    <title>Page Title</title>
  </head>
</html>

Use Semantic Elements

✅ Do This:

<header>
  <nav>
    <ul>
      <li><a href="#">Home</a></li>
    </ul>
  </nav>
</header>

❌ Avoid This:

<div class="header">
  <div class="nav">
    <div><a href="#">Home</a></div>
  </div>
</div>

Coding Style

  • Use lowercase for element names and attributes
  • Use double quotes for attribute values
  • Indent nested elements consistently
  • Keep lines under 80 characters when possible
  • Use meaningful class and ID names

Examples

✅ Good Style:

<article class="blog-post">
  <h1 class="post-title">Title</h1>
  <p class="post-content">Content</p>
</article>

❌ Poor Style:

<ARTICLE CLASS="p1">
<H1 CLASS='t1'>Title</H1>
<P CLASS='c1'>Content</P>
</ARTICLE>

Accessibility

Proper Heading Structure

✅ Do This:

<h1>Main Title</h1>
<section>
  <h2>Section Title</h2>
  <h3>Subsection</h3>
</section>

❌ Avoid This:

<h3>Main Title</h3>
<section>
  <h1>Section Title</h1>
  <h4>Subsection</h4>
</section>

Form Labels and ARIA

✅ Do This:

<label for="name">Name:</label>
<input 
  id="name"
  type="text"
  aria-required="true"
  aria-invalid="false"
>

❌ Avoid This:

<input type="text" placeholder="Enter name">

Performance

Image Optimization

  • Use appropriate image formats (WebP, AVIF for modern browsers)
  • Provide responsive images with srcset
  • Lazy load images below the fold
  • Include width and height attributes
<img
  src="small.jpg"
  srcset="small.jpg 300w, medium.jpg 600w, large.jpg 900w"
  sizes="(max-width: 600px) 300px, (max-width: 900px) 600px, 900px"
  loading="lazy"
  width="800"
  height="600"
  alt="Description"
>

Script Loading

<!-- Defer non-critical scripts -->
<script src="main.js" defer></script>

<!-- Async for independent scripts -->
<script src="analytics.js" async></script>

SEO Best Practices

  • Use descriptive title tags
  • Include meta descriptions
  • Structure content with semantic headings
  • Use descriptive link text
  • Implement proper Open Graph tags
<head>
  <title>Clear, Descriptive Page Title | Site Name</title>
  <meta name="description" content="Clear, compelling description">
  <meta property="og:title" content="Page Title">
  <meta property="og:description" content="Page description">
  <meta property="og:image" content="share-image.jpg">
</head>

Security Considerations

  • Use HTTPS for all resources
  • Add appropriate security headers
  • Sanitize user input
  • Use Content Security Policy (CSP)
  • Implement cross-origin attributes properly
<!-- External resources -->
<img crossorigin="anonymous" src="https://example.com/image.jpg">

<!-- Forms -->
<form method="post" action="/submit">
  <input type="text" pattern="[a-zA-Z0-9]+" required>
</form>

Common Mistakes to Avoid

  • Using tables for layout
  • Overusing divs instead of semantic elements
  • Missing alt attributes on images
  • Improper heading hierarchy
  • Relying on color alone for meaning
  • Using inline styles
  • Omitting required attributes
