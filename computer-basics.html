Understanding Computer, Internet, and Developer Tooling

Computer Hardware Basics

Motherboard

The main circuit board that holds all components and facilitates communication between them. It connects memory, processors, storage devices, and other essential hardware.

CPU (Central Processing Unit)

The processor responsible for executing instructions and performing calculations. Acts as the computer's "brain".

RAM (Random Access Memory)

Temporary storage location for active programs and data that the CPU needs to access quickly.

Storage Devices

  • HDD (Hard Disk Drive): Traditional storage using magnetic disks for permanent data storage
  • SSD (Solid State Drive): Faster storage using non-volatile flash memory

PSU (Power Supply Unit)

Converts AC power from wall outlets into appropriate DC power for computer components.

GPU (Graphics Processing Unit)

Specialized processor for rendering visuals and handling graphics-intensive tasks.

Internet and Connectivity

Internet Service Providers (ISPs)

  • Dial-up: Legacy connection using telephone lines
  • DSL: Digital Subscriber Line using telephone infrastructure
  • Cable: High-speed internet through cable TV infrastructure
  • Fiber-optic: Ultra-fast connections using light through fiber cables
  • Satellite: Internet access via orbital satellites

Computer Security

Safe Authentication Practices

  • Use strong, unique passwords
  • Enable two-factor authentication (2FA)
  • Utilize password managers
  • Regular security updates

Developer Tools and Environment

Development Tools

IDEs and Editors

  • IDE (Integrated Development Environment): Comprehensive development tools
  • Code Editor: Lightweight tool for writing and editing code

Version Control

  • Git: Popular version control system
  • Cloud Hosting: GitHub, GitLab, Bitbucket

Package Managers

  • npm (Node.js)
  • pip (Python)
  • Maven (Java)

Testing Tools

  • Jest (JavaScript)
  • PHPUnit (PHP)
  • JUnit (Java)

Working With Files and Media

File Management

  • Naming Best Practices: Use descriptive names (e.g., about-us.html vs page1.html)
  • Root Directory: Top-level folder in a file system
  • index.html: Default webpage for websites

Common File Formats

Web Development

  • .html (HTML files)
  • .css (Stylesheet files)
  • .js (JavaScript files)
  • .md (Markdown documentation)

Images and Graphics

  • .jpg/.jpeg (Photos)
  • .png (Graphics with transparency)
  • .gif (Animated images)
  • .svg (Vector graphics)

Media Files

  • .mp3 (Audio)
  • .mp4 (Video)
  • .mov (Apple video format)

Fonts

  • .ttf (TrueType fonts)
  • .woff (Web fonts)
  • .woff2 (Compressed web fonts)

Web Browsing Fundamentals

Core Concepts

  • Web Browser: Software for accessing and viewing websites
  • Search Engine: Tool for finding information online

Popular Browsers

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

Advanced Search Techniques

  • site:example.com - Search within a specific website
  • filetype:pdf - Search for specific file types
  • -term - Exclude results containing a term
  • +term - Must include this term

Knowledge Check

Test your understanding of computer basics and developer tools with these key concepts:

Computer Hardware

Basic Computer Components

Q: Which of the following is a basic part of the computer?

A: The central processing unit (CPU) - The "brain" of the computer that processes instructions

Q: What is random access memory?

A: A temporary storage location for the computer's central processing unit

Q: What is a pointing device for a computer?

A: A mouse or touchpad - Devices that control cursor movement

Internet and Connectivity

Q: What is a type of ISP provider?

A: Tier 1 - A major internet backbone provider (Note: Fiber optic, Satellite, and Dial-up are connection types, not provider classifications)

Q: What is a search engine?

A: A web-based tool for searching information on the internet

File Management

Q: What is a best practice when naming files for a web application?

A: Using hyphens instead of spaces (e.g., my-file.html instead of my file.html)

Q: What is the default page for a website?

A: index.html

Q: Which file managers can you use to create, move, and delete files?

A: Explorer (Windows) or Finder (macOS)

Q: What is the built-in way you can use to search for files on your computer?

A: The file explorer

File Types and Formats

Q: Which of the following is a common file type when working with web applications?

A: .css - Used for styling web pages

Q: What is a common file type for a video file?

A: .mp4 - A widely supported video format

Q: Which file type is commonly used for image files?

A: .svg - Scalable Vector Graphics, perfect for logos and icons

Q: Which file format is used to compress files?

A: ZIP - A common compression format

Developer Tools and Browsers

Q: What is Git?

A: A version control system for tracking changes and collaborating with others

Q: Which of the following is NOT a cloud-based hosting service for repositories?

A: Gitpod (Common hosting services include GitHub, GitLab, and Bitbucket)

Q: Which of the following is not primarily a part of an integrated development environment?

A: A tool for storing code (IDEs are for writing, testing, and debugging code)

Q: Microsoft Edge is supported by which operating system?

A: Windows (though it's now available on other platforms too)

Q: Which of the following is NOT a type of browser?

A: GitHub (GitHub is a development platform, not a browser)

Search Strategies

Q: What is a common search strategy to search for file types?

A: filetype: (e.g., filetype:pdf to search for PDF files)
