Understanding Computer, Internet, and Developer Tooling

Computer Hardware Basics

Motherboard

The main circuit board that holds all components and facilitates communication between them. It connects memory, processors, storage devices, and other essential hardware.

CPU (Central Processing Unit)

The processor responsible for executing instructions and performing calculations. Acts as the computer's "brain".

RAM (Random Access Memory)

Temporary storage location for active programs and data that the CPU needs to access quickly.

Storage Devices

  • HDD (Hard Disk Drive): Traditional storage using magnetic disks for permanent data storage
  • SSD (Solid State Drive): Faster storage using non-volatile flash memory

PSU (Power Supply Unit)

Converts AC power from wall outlets into appropriate DC power for computer components.

GPU (Graphics Processing Unit)

Specialized processor for rendering visuals and handling graphics-intensive tasks.

Internet and Connectivity

Internet Service Providers (ISPs)

  • Dial-up: Legacy connection using telephone lines
  • DSL: Digital Subscriber Line using telephone infrastructure
  • Cable: High-speed internet through cable TV infrastructure
  • Fiber-optic: Ultra-fast connections using light through fiber cables
  • Satellite: Internet access via orbital satellites

Computer Security

Safe Authentication Practices

  • Use strong, unique passwords
  • Enable two-factor authentication (2FA)
  • Utilize password managers
  • Regular security updates

Developer Tools and Environment

Development Tools

IDEs and Editors

  • IDE (Integrated Development Environment): Comprehensive development tools
  • Code Editor: Lightweight tool for writing and editing code

Version Control

  • Git: Popular version control system
  • Cloud Hosting: GitHub, GitLab, Bitbucket

Package Managers

  • npm (Node.js)
  • pip (Python)
  • Maven (Java)

Testing Tools

  • Jest (JavaScript)
  • PHPUnit (PHP)
  • JUnit (Java)

Working With Files and Media

File Management

  • Naming Best Practices: Use descriptive names (e.g., about-us.html vs page1.html)
  • Root Directory: Top-level folder in a file system
  • index.html: Default webpage for websites

Common File Formats

Web Development

  • .html (HTML files)
  • .css (Stylesheet files)
  • .js (JavaScript files)
  • .md (Markdown documentation)

Images and Graphics

  • .jpg/.jpeg (Photos)
  • .png (Graphics with transparency)
  • .gif (Animated images)
  • .svg (Vector graphics)

Media Files

  • .mp3 (Audio)
  • .mp4 (Video)
  • .mov (Apple video format)

Fonts

  • .ttf (TrueType fonts)
  • .woff (Web fonts)
  • .woff2 (Compressed web fonts)

Web Browsing Fundamentals

Core Concepts

  • Web Browser: Software for accessing and viewing websites
  • Search Engine: Tool for finding information online

Popular Browsers

  • Google Chrome
  • Mozilla Firefox
  • Microsoft Edge
  • Safari

Advanced Search Techniques

  • site:example.com - Search within a specific website
  • filetype:pdf - Search for specific file types
  • -term - Exclude results containing a term
  • +term - Must include this term
