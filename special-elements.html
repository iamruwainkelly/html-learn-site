Special Elements and Characters

HTML Entities

HTML entities are used to display reserved characters and symbols that might otherwise be interpreted as HTML markup.

Common HTML Entities

Character Entity Name Entity Number Description
< &lt; &#60; Less than
> &gt; &#62; Greater than
& &amp; &#38; Ampersand
" &quot; &#34; Double quote
' &apos; &#39; Single quote

Text-Level Semantic Elements

Emphasis Elements

<!-- Emphasis -->
<em>This text has emphasis</em>

<!-- Strong importance -->
<strong>This text is strongly emphasized</strong>

<!-- Marked/highlighted text -->
<mark>This text is highlighted</mark>

Technical and Special Text

<!-- Code -->
<code>console.log('Hello');</code>

<!-- Keyboard input -->
<kbd>Ctrl</kbd> + <kbd>C</kbd>

<!-- Sample output -->
<samp>Error: File not found</samp>

<!-- Variables -->
<var>x</var> = <var>y</var> + 2

Ruby Annotations

<ruby>
  漢 <rt>かん</rt>
  字 <rt>じ</rt>
</ruby>

Time and Date

<!-- Date -->
<time datetime="2025-05-16">May 16, 2025</time>

<!-- Time -->
<time datetime="14:30">2:30 PM</time>

<!-- DateTime -->
<time datetime="2025-05-16T14:30:00+00:00">
  May 16, 2025, 2:30 PM UTC
</time>

Citations and Quotes

Block Quotes

<blockquote cite="https://example.com/quote">
  <p>This is a long block quote that might span multiple paragraphs.</p>
  <footer>
    — <cite>Author Name</cite>
  </footer>
</blockquote>

Inline Quotes

<p>The instructor said, <q>Always use semantic HTML</q></p>

Description Lists

<dl>
  <dt>HTML</dt>
  <dd>HyperText Markup Language</dd>
  
  <dt>CSS</dt>
  <dd>Cascading Style Sheets</dd>
</dl>

Text Directionality

<!-- Right-to-left text -->
<p dir="rtl">This text flows right to left</p>

<!-- Bidirectional text -->
<p>English text <bdi>العربية</bdi> more English</p>

<!-- Bidirectional isolation -->
<bdo dir="rtl">This text will go right to left</bdo>
